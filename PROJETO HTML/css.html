C++

C++ é uma linguagem de programação que pode ser considerada uma melhoria da linguagem C. Assim, nesta evolução, recebeu a possibilidade de trabalhar com a característica de orientação a objetos. Foi desenvolvida em 1983, pelo cientista da computação dinamarquês e professor catedrático da Universidade do Texas Bjarne Stroustrup, que hoje é considerado o pai da linguagem C++. Por ser muito utilizada, C++ continua sua evolução e se destaca também no desenvolvimento de softwares para desktop e de games.

C++ é uma das linguagens mais versáteis que existem, permitindo desenvolver desde tarefas simples como aplicações na linha de comando ou web, até sistemas complexos de tempo real, muito usadas no mercado financeiro.

CSS

CSS é a sigla para o termo em inglês Cascading Style Sheets que, traduzido, significa Folha de Estilo em Cascatas. Fácil de aprender, o CSS é mais utilizado com as linguagens de marcação HTML e XHTML. Na prática, é um mecanismo para adicionar estilo (cores, fontes, espaçamento, responsividade, etc.) a uma página na web. Mas, no lugar de formatar dentro do próprio documento, o CSS cria um link que direciona a outro arquivo. Este arquivo contém todos os estilos utilizados no site. A primeira versão do CSS foi proposto em 1994, por Håkon Wium Lie, para controlar aparências de documentos em HTML. Mas, em 1995, a empresa W3C se interessou pelo projeto e, em 1996, lançaram a recomendação oficial pelo W3C do CSS nível 1 (CSS 1). Dois anos depois (1998), foi lançada a recomendação do CSS nível 2. A segunda versão das Folhas de Estilo para web. Atualmente, o que está sendo utilizado é o CSS3, o nível 3 do CSS.
