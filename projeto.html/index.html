    • Programação entre mulheres

    O objetivo da programação entre mulheres é criar espaços seguros e inclusivos para que mulheres possam aprender programação e tecnologia, trocar experiências e ideias, criar redes de contatos, e se desenvolverem profissionalmente. As iniciativas podem ser desde cursos presenciais ou online, comunidades online ou eventos presenciais, sempre com o intuito de criar oportunidades para que mulheres possam se conectar e se desenvolver.

    Linguagens de programação

    linguagens de programação são conjuntos de regras sintáticas e semânticas usadas para criar software e sistemas computacionais. Existem várias linguagens de programação, cada uma com suas próprias características e finalidades.
